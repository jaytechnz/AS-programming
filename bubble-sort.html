Bubble Sort — Code Reference

Switch between Java and Cambridge Pseudocode. Explore efficient versus inefficient implementations across different array types.

Algorithm Overview: Bubble sort repeatedly compares adjacent elements, swapping them if they are out of order. The inefficient version always performs (n−1)² comparisons using two FOR loops. The efficient version uses a swapped flag to exit early when no swaps occur in a pass.
