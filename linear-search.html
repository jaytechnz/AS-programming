Linear Search — Code Reference

Switch between Java and Cambridge Pseudocode. Explore three different approaches across different array types.

🔍 Find First

Searches through the array and exits as soon as the target value is found. Returns the index, or −1 if not found. Uses a WHILE loop in pseudocode or a for+break in Java.

🔢 Count All

Iterates through the entire array without stopping, incrementing a counter each time the target value is found. Returns the total number of occurrences. Uses a FOR loop.

📋 2D Search

Uses nested loops to search a two-dimensional array row by row and column by column. Exits as soon as the target is found, recording both the row and column index.

Key difference — Find First versus Count All: Finding the first occurrence uses early exit (the search stops the moment a match is found), making it more efficient when the value exists near the start. Counting all occurrences must visit every element, so the loop always runs to completion regardless of how many matches exist.
